\doxysubsection{Detailed Description}
\subsubsection*{template$<$int Dimension = 1, bool Delete\+Mode = true, char Symbol = \textquotesingle{}\#\textquotesingle{}, unsigned int Max\+Hashes = 20$>$\newline
class JSL\+::\+Progress\+Bar$<$ Dimension, Delete\+Mode, Symbol, Max\+Hashes $>$}
A neat display widget for tracking the progress of a process by printing a progress bar on the screen, which updates to visually indicate the process is running. Supports arbitrary \char`\"{}levels\char`\"{} of progress bars which can be independently modified through a variadic argument interface. 
\begin{DoxyTemplParams}{Template Parameters}
{\em Dimension} & The number of concurrent bars to print (fixes the number of arguments which must be provided to subsequent functions). Defaults to 1 if 0 template arguments provided \\
\hline
{\em Delete\+Mode} & If true, utilises \mbox{\hyperlink{namespaceJSL_ae38780f94d60045b904b289eb3ab8eda}{JSL\+::delete\+Line()}} to remove the previous line before reprinting, such that the bar appears animated. If false, provides a more limited form of animation (false only available in Dimension==1 mode). Defaults to true. \\
\hline
{\em Symbol} & The symbol which is printed within the bar to fill up space. Defaults to \char`\"{}\#\char`\"{}. \\
\hline
{\em Max\+Hashes} & The number of symbols in a full bar (dictates the width of the progress bar) \\
\hline
\end{DoxyTemplParams}
