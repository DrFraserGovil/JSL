Construct a persistent antenna which listens for incoming messages. 


\begin{DoxyParams}{Parameters}
{\em identifier} & A string which identifies the object; those wanting to send messages to this socket need to know this name in order to communicate. The socket is hosted at a location on disk (usually /tmp/\mbox{[}identifier\mbox{]}). If an \doxylink{classJSL_1_1Event_1_1Antenna}{Antenna} with this name already exists, the system will check if there is an active process using it\+: if so, returns nullopt. If the socket is \textquotesingle{}stale\textquotesingle{}, it is taken over by this object. \\
\hline
{\em force\+Acquire} & If true, does not return nullopt on encountering an occupied socket. Instead, issues a \textquotesingle{}shutdown\textquotesingle{} command to the existing socket, and then takes over. \\
\hline
{\em grace\+Period} & The time (in ms) allowed for a process to shutdown before the force\+Acquire protocol takes place \\
\hline
\end{DoxyParams}
