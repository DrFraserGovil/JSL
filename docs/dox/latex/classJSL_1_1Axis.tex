\doxysubsection{Detailed Description}
An axis is a single instance of a plotting `grid' on the screen. Multiple such axes can exist simultaneously (via \mbox{\hyperlink{classJSL_1_1gnuplot_aa1e759e1965581b7c4fe508d2bca7060}{gnuplot\+::\+Set\+Multiplot()}}), with each axis tracking their own data and properties individually.
