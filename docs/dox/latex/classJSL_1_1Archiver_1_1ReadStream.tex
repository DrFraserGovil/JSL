Parse the datafile and convert each line into a tuple of datatypes based on an assumed regular tabular. Then performs a callback function on each line. 

Each line is converted into a tuple of types specified by the Column\+Types parameter (i.\+e. For\+Tabular\+Line\+In$<$int,int,int$>$ attempts to read the file as a set of three integers). Tuples are then accessed via std\+::get$<$i$>$ to get the ith element of the tuple.
